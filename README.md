# ComputorV1

Program that solves a polynomial equation of degree less than or equal to 2.

Example :<br/>
"5 * X^0 + 4 * X^1 - 9.3 * X^2 = 1 * X^0"<br/>
"5 * X^0 + 4 * X^1 = 4 * X^0"<br/>
"8 * X^0 - 6 * X^1 + 0 * X^2 - 5.6 * X^3 = 3 * X^0"<br/>
"5 + 4 * X + X^2 = X^2" (Natural form)<br/>
